주요 기능 개선

글로벌 유니크 닉네임 시스템 도입

  • 리더보드 닉네임 중복 제거: 리더보드에서 닉네임은 고유 식별자로 사용되며, 게임 전체에서 중복 불가
  • 닉네임 정책 강화: 길이 1~6자, 공백 불허, 금칙어 필터 적용, 변경 쿨타임 30초
  • 닉네임 회수 정책: 계정 탈퇴 시 닉네임이 자동으로 해제되어 다른 사용자가 사용할 수 있게 됩니다

닉네임 변경 UX 개선

  • 모달 방식으로 전환, 자동 포커스 및 전체 선택
  • 중복 렌더링 버그 수정
  • 마이그레이션 충돌 UX 강화

업적 영역 스크롤 최적화

  • 접기 기능 제거로 높이 변동으로 인한 스크롤 튐 현상 해결
  • 높이 안정화 및 스크롤 성능 개선

초기 릴리즈

  • 브라우저 기반 자본 축적 클리커 게임 (Cookie Clicker 감성 + 서울 생존 테마)
  • 다국어 지원: 한국어/영어
  • 소셜 로그인(SSO): Google 로그인으로 허브와 게임 간 동일 로그인 상태 공유
  • 클라우드 저장: 로그인 사용자는 탭 숨김/닫기 시 자동 클라우드 저장
  • 리더보드: 닉네임 기반 TOP 10 리더보드 (총 자산, 플레이타임, 서울타워 개수)
  • 프레스티지 시스템: 서울타워 구매 시 엔딩 모달 표시 및 자동 프레스티지
  • 노동/커리어: 알바 → CEO (직급별 야간 도트 배경)
  • 투자: 예금 → 적금 → 국내주식 → 미국주식 → 코인 → 빌라 → … → 빌딩 → 서울타워
  • 업그레이드: 해금/구매 기반 업그레이드 시스템
  • 반응형 UI: PC/태블릿/모바일 지원
