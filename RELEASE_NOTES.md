## v1.2.2 (2025-12-31)

### MMA Manager (서브게임) 긴급 수정

- **[CRITICAL] 닉네임 렌더링 크래시 수정**: 파이터 닉네임 데이터 구조 변경(`string` -> `{ko, en}`)으로 인해 발생하던 화면 깨짐/크래시 버그를 수정했습니다.
- **안전한 닉네임 표기**: 구버전 데이터와 신버전 데이터를 모두 안전하게 처리하는 공용 헬퍼(`ui/utils/fighterName.ts`) 도입.

## v1.2.0 (2025-12-25)

### 주요 기능 개선

#### 글로벌 유니크 닉네임 시스템 도입

- **리더보드 닉네임 중복 제거**: 리더보드에서 닉네임은 고유 식별자로 사용되며, 게임 전체에서 중복 불가
- **닉네임 정책 강화**:
  - 길이: 1~6자 (기존 2~12자에서 변경)
  - 공백 불허 (기존 공백 허용에서 변경)
  - 금칙어 필터 적용 (욕설/비하/시스템 키워드 포함 닉네임 사용 불가)
  - 변경 쿨타임: 30초 (스팸 방지)
- **닉네임 회수 정책**: 계정 탈퇴 시 닉네임이 자동으로 해제되어 다른 사용자가 사용할 수 있게 됩니다
- **마이그레이션**: 기존 사용자 중 일부는 닉네임이 중복되어 변경이 필요할 수 있습니다
  - 로그인 시 자동으로 닉네임 중복 여부를 확인합니다
  - 중복이 감지되면 설정 탭에서 닉네임 변경을 안내합니다

#### 닉네임 변경 UX 개선

- **모달 방식으로 전환**: 상시 입력창에서 버튼 클릭 → 모달 입력 방식으로 변경
- **자동 포커스 및 전체 선택**: 모달 오픈 시 입력창에 자동 포커스 및 전체 선택
- **중복 렌더링 버그 수정**: 모달 재오픈 시 설명 문구 중복 누적 문제 해결
- **마이그레이션 충돌 UX 강화**: 중복 닉네임 감지 시 자동으로 입력 모달 오픈

#### 업적 영역 스크롤 최적화

- **접기 기능 제거**: 업적 영역의 접기/펼치기 기능을 제거하여 높이 변동으로 인한 스크롤 튐 현상 해결
- **높이 안정화**: `min-height: clamp(320px, 45vh, 520px)` 적용으로 초기 렌더/데이터 변화 시 높이 흔들림 방지
- **스크롤 성능 개선**: CSS 최적화 속성 적용 (`overscroll-behavior: contain`, `overflow-anchor: none`, `contain: layout paint`)

### 기술적 변경사항

- `nickname_registry` 테이블 추가 (Supabase SQL 실행 필요)
- 닉네임 클레임/회수 RPC 함수 추가
- 닉네임 정규화 시스템 개선 (NFC + 대소문자 구분 없음)
- 버전 관리 시스템 개선: `package.json`을 source of truth로 사용하여 게임 내 버전 표시 자동 동기화

## v1.0.0 (2025-12-21)

### 초기 릴리즈

- 브라우저 기반 자본 축적 클리커 게임 (Cookie Clicker 감성 + 서울 생존 테마)
- 다국어 지원: 한국어/영어
- 소셜 로그인(SSO): Google 로그인으로 허브와 게임 간 동일 로그인 상태 공유
- 클라우드 저장: 로그인 사용자는 탭 숨김/닫기 시 자동 클라우드 저장
- 리더보드: 닉네임 기반 TOP 10 리더보드 (총 자산, 플레이타임, 서울타워 개수)
- 프레스티지 시스템: 서울타워 구매 시 엔딩 모달 표시 및 자동 프레스티지
- 노동/커리어: 알바 → CEO (직급별 야간 도트 배경)
- 투자: 예금 → 적금 → 국내주식 → 미국주식 → 코인 → 빌라 → … → 빌딩 → 서울타워
- 업그레이드: 해금/구매 기반 업그레이드 시스템
- 반응형 UI: PC/태블릿/모바일 지원
