# Capital Clicker 💰

브라우저 기반 자본 축적 클리커 게임 (Cookie Clicker 스타일)

## 🎮 게임 소개

클릭으로 돈을 벌고, 금융상품과 부동산에 투자하여 자본을 증식시키는 클리커 게임입니다. 노동으로 시작해 CEO가 되고, 다양한 자산을 구매하여 부를 쌓아보세요!

### 주요 특징
- 🛠️ **노동 시스템**: 클릭으로 돈을 벌고 커리어를 쌓으세요 (알바→CEO)
- 💰 **금융상품**: 예금, 적금, 국내주식, 미국주식, 코인으로 자산 증식
- 🏘️ **부동산 투자**: 빌라부터 빌딩까지 단계적 성장
- 🎨 **직급별 배경**: 알바부터 CEO까지 10개 직급별 고유 야간 도트 배경 이미지
- 🔓 **순차 해금**: 예금→적금→국내주식→미국주식→코인→빌라→...빌딩 순서로 해금
- 📈 **업그레이드**: 48개 재밸런싱된 업그레이드 시스템
- 📊 **통계 시스템**: 누적 수익, 효율 분석, 32개 업적 그리드
- ⚡ **시장 이벤트**: 상품별 세분화된 시장 이벤트 (호황/침체)
- ⚖️ **단순한 수익 구조**: 초당 수익은 고정 수익률 기반(변동은 시장 이벤트로만 표현)
- 💾 **자동 저장**: 진행상황 자동 저장 (5초마다)
- 📱 **완벽한 반응형**: PC, 태블릿, 모바일 모두 최적화
- 🎨 **시각 효과**: 떨어지는 지폐/상품 애니메이션, 승진 시 배경 전환

### 초기 설정 (1회만 실행)

1. GitHub에서 새 저장소 생성
2. 다음 명령어로 원격 저장소 연결:
```bash
git remote add origin https://github.com/sangbal/저장소명.git
git push -u origin main
```

3. GitHub 저장소 설정에서 Pages 활성화:
   - Settings → Pages
   - Source: "Deploy from a branch"
   - Branch: "main" / "/ (root)"
   - Save


#### 방법 2: PowerShell 스크립트 사용
```bash
.\deploy.ps1
```

#### 방법 3: 수동 명령어

## 📊 게임 시스템

### 수익 구조
- **노동 (클릭)**: 알바(1만원) → CEO(25만원), 업그레이드로 최대 285만원/클릭
- **금융상품**: 예금(10원/초) → 적금(120원/초) → 국내주식(1,125원/초) → 미국주식(6,000원/초) → 코인(25,000원/초)
- **부동산**: 빌라(250원/초) → 빌딩(5,000원/초), 초당 수익
- **시장 이벤트**: 상품별 세분화된 이벤트 (강남 아파트 대박, 코인 시장 폭락 등)
- **수익 변동**: 시장 이벤트에 의해서만 일시적으로 변화

### 가격 시스템
- **구매**: 각 상품 구매 시 1.15배 증가 (Cookie Clicker 방식)
- **판매**: 현재가의 80% 회수
- **수량 선택**: 1개, 5개, 10개 단위 거래

### 직급 시스템 (10단계)
1. 알바 (1만원/클릭)
2. 계약직 (1.5만원/클릭)
3. 사원 (2만원/클릭)
4. 대리 (2.5만원/클릭)
5. 과장 (3만원/클릭)
6. 차장 (3.5만원/클릭)
7. 부장 (4만원/클릭)
8. 상무 (5만원/클릭)
9. 전무 (10만원/클릭)
10. CEO (25만원/클릭)

각 직급마다 고유한 야간 도트 배경 이미지가 표시되며, 승진 시 부드러운 페이드 전환 애니메이션이 적용됩니다.

### 순차 해금 시스템
1. 예금 (항상 가능) → 2. 적금 → 3. 국내주식 → 4. 미국주식 → 5. 코인
6. 빌라 → 7. 오피스텔 → 8. 아파트 → 9. 상가 → 10. 빌딩
- 이전 단계 1개 보유 시 자동 해금
- 🔒 잠김 상태 시각화 (회색 + 잠금 아이콘)
- 중반 허리 이어주기: 미국주식과 코인으로 부드러운 진행 곡선

### 업그레이드 시스템 (48개)
- **노동 업그레이드**: 클릭 효율 증가 (×1.5, ×1.3, ×2 등)
- **금융 업그레이드**: 예금/적금/주식 각각 2배 증가
- **부동산 업그레이드**: 각 부동산 종류별 2배 증가
- **전역 업그레이드**: 모든 부동산 +10%, 자동 클릭 등
- **해금 조건**: 클릭 수, 보유 개수, 누적 수익 기반

### 업적 시스템 (32개)
- 노동 관련 업적 (직급 달성, 클릭 횟수 등)
- 자산 관련 업적 (보유 개수, 누적 수익 등)
- 시장 이벤트 관련 업적
- 특별 업적 (전설의 CEO 등)
