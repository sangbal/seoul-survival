# Step 4 완료 리포트: 허브홈 페이지 생성 (v0.1) - 헤더 계정 버튼 추가

## 작업 내용

### 1. 헤더 계정 버튼 추가
- **위치**: `shared/shell/header.js`
- **기능**:
  - 로그인 안 됨: "Login" 버튼 표시
  - 로그인 됨: 닉네임(텍스트) + 드롭다운 메뉴 (계정 관리, 로그아웃)
- **스타일**: SeoulSurvival 패턴의 `.chip` 스타일 사용

### 2. 계정 드롭다운 메뉴
- **위치**: `shared/styles/header.css`
- **기능**:
  - 계정 관리 링크 (`./account/`)
  - 로그아웃 버튼
  - 외부 클릭 시 자동 닫힘
- **반응형**: 모바일에서 바텀시트 스타일로 변경

### 3. 인증 UI 초기화
- **위치**: `hub/main.js`
- **기능**:
  - `initHeaderAuth()` 함수로 헤더 계정 버튼 인증 상태 관리
  - 로그인 상태 변경 시 UI 자동 업데이트
  - 게스트 모드 기본 상태 설정

### 4. 빌드 설정 수정
- **위치**: `vite.config.js`
- **변경**: `seoulsurvival/index.html`을 빌드 입력에 추가하여 `/seoulsurvival/` 경로 접근 가능
