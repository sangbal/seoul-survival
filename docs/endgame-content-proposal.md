# 엔드게임 컨텐츠 기획안: Capital Clicker Seoul Survival

## Executive Summary

현재 게임은 CEO 달성(15,000 클릭) 이후 명확한 목표 부재로 장기 리텐션 저하 위험이 존재한다. 유사 성공 게임들의 엔드게임 메커니즘 분석 결과, **프레스티지/메타 진행 시스템** 도입이 핵심 해결책으로 도출되었다. 본 기획안은 3단계 엔드게임 컨텐츠 로드맵을 제시하며, 기존 게임플레이와의 호환성을 유지하면서 점진적 확장을 제안한다.

---

## 핵심 질문

1. **CEO 달성 후 플레이어가 지속적으로 돌아올 동기**는 무엇인가?
2. **기존 진행 상황을 유지하면서 새로운 목표**를 제공할 수 있는가?
3. **단순 반복이 아닌 전략적 깊이**를 추가할 수 있는가?

---

## 전략 방향

### 1. 벤치마킹 분석 결과

| 게임 | 엔드게임 메커니즘 | 핵심 성공 요인 |
|------|-----------------|---------------|
| **Cookie Clicker** | Prestige (Heavenly Chips), Ascension, Sugar Lumps, Garden, Stock Market | 환생 보상이 다음 플레이를 더 빠르게 만듦, 여러 병렬 시스템으로 선택권 제공 |
| **Adventure Capitalist** | Prestige (Angels), Moon/Mars 확장, 이벤트 | 환생 시 영구 배수 획득, 새로운 지역으로 확장 |
| **Idle Games 일반** | Meta Progression, Prestige Layers, Permanent Upgrades | 환생 시 영구 보너스, 다음 사이클이 더 빠르게 진행 |

### 2. 핵심 설계 원칙

- **기존 진행 보존**: 환생 시에도 업적/통계는 누적 유지
- **점진적 해금**: CEO 달성 후 단계적으로 엔드게임 컨텐츠 노출
- **선택의 자유**: 여러 엔드게임 경로 중 선택 가능
- **영구 보상**: 환생 시 획득한 보너스는 다음 플레이에 지속 적용

## 주요 과제

### Phase 1: 프레스티지 시스템 (Prestige System)

#### 1.1 환생 메커니즘

**개념**: CEO 달성 후 일정 자산 달성 시 "환생" 가능. 환생 시 현재 진행을 초기화하되, **영구 보너스**를 획득하여 다음 플레이가 더 빠르게 진행됨.

**해금 조건**:
- CEO 달성 (`careerLevel >= 9`)
- 총 자산 1조원 달성 (`totalAssets >= 1,000,000,000,000`)

**환생 보상 계산**:
```
영구 자본 (Permanent Capital) = sqrt(총 획득 자산 / 1조) × 100
```

**영구 보너스**:
- 노동 수익 배수: `1 + (영구 자본 × 0.1%)`
- 금융 수익 배수: `1 + (영구 자본 × 0.15%)`
- 부동산 수익 배수: `1 + (영구 자본 × 0.2%)`
- 초기 현금 보너스: `영구 자본 × 1억원`

**UI/UX**:
- 설정 탭에 "🔄 환생" 섹션 추가
- 환생 가능 여부 표시 (조건 미달 시 남은 자산 표시)
- 환생 시 획득할 영구 자본 미리보기
- 환생 확인 다이얼로그 (2단계 확인)

#### 1.2 프레스티지 업그레이드

환생 시 획득한 영구 자본으로 구매 가능한 **영구 업그레이드** 시스템.

| 업그레이드 ID | 이름 | 비용 (영구 자본) | 효과 |
|-------------|------|----------------|------|
| `prestige_work_boost_1` | 노동 효율 강화 I | 10 | 노동 수익 +5% |
| `prestige_work_boost_2` | 노동 효율 강화 II | 25 | 노동 수익 +10% |
| `prestige_financial_boost_1` | 금융 전문성 I | 15 | 금융 수익 +7% |
| `prestige_financial_boost_2` | 금융 전문성 II | 35 | 금융 수익 +15% |
| `prestige_property_boost_1` | 부동산 경험 I | 20 | 부동산 수익 +10% |
| `prestige_property_boost_2` | 부동산 경험 II | 50 | 부동산 수익 +20% |
| `prestige_starting_cash` | 초기 자본 | 30 | 환생 시 추가 현금 10억원 |
| `prestige_unlock_speed` | 빠른 해금 | 40 | 순차 해금 조건 50% 감소 |

**저장 구조**:
```javascript
{
  prestigeLevel: 0,           // 환생 횟수
  permanentCapital: 0,        // 현재 보유 영구 자본
  totalPermanentCapital: 0,   // 누적 획득 영구 자본
  prestigeUpgrades: {}        // 구매한 프레스티지 업그레이드
}
```

### Phase 2: 확장 지역 시스템 (Expansion Regions)

#### 2.1 지역 해금 구조

CEO 달성 후 새로운 투자 지역이 순차적으로 해금됨.

| 지역 | 해금 조건 | 특징 |
|------|----------|------|
| **강남** | CEO + 자산 5조 | 아파트/상가 수익 +20% |
| **제주** | 강남 해금 + 자산 10조 | 리조트 수익 (새 상품) |
| **부산** | 제주 해금 + 자산 20조 | 항구/물류 수익 (새 상품) |
| **해외** | 부산 해금 + 자산 50조 | 글로벌 투자 확장 |

#### 2.2 지역별 특수 상품

각 지역마다 고유한 투자 상품 추가.

**강남**:
- 강남 오피스텔: 기본 오피스텔 수익 ×1.5
- 강남 상가: 기본 상가 수익 ×1.3

**제주**:
- 리조트: 초당 수익 200만원, 가격 50억
- 콘도: 초당 수익 80만원, 가격 15억

**부산**:
- 항구 창고: 초당 수익 500만원, 가격 100억
- 물류 센터: 초당 수익 300만원, 가격 60억

**해외**:
- 뉴욕 부동산: 초당 수익 1,000만원, 가격 200억
- 런던 부동산: 초당 수익 800만원, 가격 150억

### Phase 3: 메타 진행 시스템 (Meta Progression)

#### 3.1 마일스톤 보상

특정 자산 달성 시 영구 보상 획득.

| 마일스톤 | 조건 | 보상 |
|---------|------|------|
| 첫 1조 | 총 자산 1조 | 영구 자본 +10 |
| 첫 10조 | 총 자산 10조 | 영구 자본 +50 |
| 첫 100조 | 총 자산 100조 | 영구 자본 +200 |
| CEO 10회 | CEO 달성 10회 | 노동 수익 영구 +5% |
| 환생 5회 | 환생 5회 | 모든 수익 영구 +10% |

#### 3.2 시즌 이벤트

정기적인 시즌 이벤트로 엔드게임 플레이어에게 추가 목표 제공.

**구조**:
- 2주마다 새로운 시즌 시작
- 시즌별 특수 목표 (예: "이번 시즌 부동산 수익 1위")
- 시즌 종료 시 보상 지급 (영구 자본, 특수 칭호 등)

**예시 시즌**:
- **부동산 시즌**: 부동산 수익이 2배
- **금융 시즌**: 금융 수익이 1.5배, 변동성 증가
- **노동 시즌**: 클릭 수익이 3배

#### 3.3 리더보드 확장

현재 리더보드를 확장하여 엔드게임 경쟁 요소 강화.

**추가 리더보드**:
- 환생 횟수 TOP 10
- 영구 자본 TOP 10
- 시즌별 랭킹 (2주마다 리셋)
- 지역별 해금 진행도 랭킹

## 밸런싱 고려사항

### 프레스티지 보상 밸런스

**목표**: 환생 후 다음 플레이가 **30~50% 빠르게** 진행되어야 함.

**조정 포인트**:
- 영구 자본 계산식의 계수 조정
- 프레스티지 업그레이드 비용/효과 재조정
- 초기 자본 보너스 크기 조정

### 확장 지역 밸런스

**원칙**: 새로운 지역 상품은 기존 최고 상품(빌딩)보다 **1.2~1.5배 효율적**이어야 해금 동기가 생김.

**가격/수익 비율**: 기존 상품과 유사한 ROI 유지 (약 0.01~0.02% 초당 수익률)

## 사용자 경험 설계

### 환생 플로우

1. **조건 달성 시 알림**: "🎉 환생 가능! 설정 탭에서 확인하세요"
2. **설정 탭 방문**: 환생 섹션에서 획득 보상 미리보기
3. **2단계 확인**: 
   - 1단계: "환생하시겠습니까?" (현재 진행 초기화 안내)
   - 2단계: "정말 환생하시겠습니까?" (최종 확인)
4. **환생 실행**: 진행 초기화 + 영구 보너스 적용
5. **환생 후 안내**: "환생 완료! 영구 자본 XX를 획득했습니다. 프레스티지 업그레이드를 확인하세요"

### 프레스티지 업그레이드 UI

- 별도 탭 또는 설정 탭 하위 섹션
- 구매 가능/불가능 시각적 구분
- 구매 시 즉시 효과 적용 안내
- 영구 자본 잔액 표시

## 기술 구현 고려사항

### 저장 호환성

- 기존 저장 데이터에 `prestige` 필드 추가 (기본값: `null` 또는 `{prestigeLevel: 0, ...}`)
- 로드 시 프레스티지 데이터가 없으면 기본값으로 초기화
- 영구 보너스는 게임 시작 시 자동 적용

### 성능

- 프레스티지 업그레이드 효과는 게임 시작 시 1회 계산하여 캐시
- 환생 시 대량 데이터 초기화는 비동기 처리 고려

## 성공 지표 (KPI)

### 리텐션 지표
- CEO 달성 후 7일 리텐션: 현재 기준선 → 목표 +30%
- CEO 달성 후 30일 리텐션: 현재 기준선 → 목표 +20%

### 참여 지표
- 환생 실행률: CEO 달성 플레이어 중 40% 이상
- 평균 환생 횟수: 2회 이상
- 프레스티지 업그레이드 구매율: 환생 플레이어 중 60% 이상

### 경쟁 지표
- 리더보드 업데이트 빈도: 일평균 +50%
- 시즌 이벤트 참여율: 활성 플레이어 중 30% 이상

## 결론

엔드게임 컨텐츠 부재 문제는 **프레스티지 시스템 도입**으로 해결 가능하다. Cookie Clicker, Adventure Capitalist 등 성공 사례를 벤치마킹하여, 기존 게임플레이와 호환되면서 장기 리텐션을 높이는 3단계 로드맵을 제시하였다.

**핵심 성공 요인**:
1. 환생 시 영구 보너스로 다음 플레이 가속화
2. 여러 병렬 시스템(지역 확장, 시즌 이벤트)으로 선택권 제공
3. 기존 진행(업적/통계) 보존으로 성취감 유지

**다음 단계**: MVP(v1.1) 프레스티지 시스템 기본 구현부터 시작하여, 사용자 피드백을 수집하며 점진적으로 확장하는 것을 권장한다.
